
# NSG Indoor Tracking System

## Problem Statement
Tracking soldiers inside buildings is difficult using traditional GPS systems. This project aims to develop a system for indoor tracking with 3D visualization.

## Features
- Real-time location tracking
- 3D building visualization
- Monitoring multiple teams
